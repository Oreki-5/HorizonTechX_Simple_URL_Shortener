# HorizonTechX_Simple_URL_Shortener

A Simple application which takes any valid and long URL and converts it into short, sharable links.

# Features:
- Convert long URLs into short links.
- Redirect user to the original URL from the generated short URL.
- Input URL validation to prevent spam inputs.
- No URL collision.
- Simple UI with copy-to-clipboard feature.

# Requirements:
- Java 21
- Node.js

# Instrctions to run

1. Clone the Repo:
```
git clone https://github.com/Oreki-5/HorizonTechX_Simple_URL_Shortener/edit/main
```
3. Change directory to backend application
```
cd Backend/UrlShortener
```
4. Build and Run the Spring boot application
```
.\mvnw clean install
.\mvnw spring-boot:run
```
6. Open the Frontend directory in another instance of command-line
```
cd Frontend/Frontend
```
6. Run the React app:
```
npm run dev
```
