# HorizonTechX_Simple_URL_Shortener

A Simple application which takes any valid and long URL and converts it into short, sharable links.

# Requirements:
* Java 21
* node.js

# Instrctions to run

1. Clone the Repo:
> git clone https://github.com/Oreki-5/HorizonTechX_Simple_URL_Shortener/edit/main

2. Change directory to backend application
> cd Backend/UrlShortener

3. Open the project in IDE like VSCode and Run the application.

4. Open the Frontend directory in another instance of command-line
> cd Frontend/Frontend

5. Run the React app:
> npm run dev
